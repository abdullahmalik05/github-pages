Frequently Asked Questions

Q1: GitHub kya hai?
GitHub ek platform hai jahan aap apne code ko store, share aur manage kar sakte hain. Yeh Git system par based hai.
Q2: HTML file kaise banayein?
Aap kisi bhi text editor (jaise Notepad, VS Code) mein `.html` extension ke saath file bana sakte hain.
Q3: Collaborator kya hota hai GitHub par?
Collaborator wo hota hai jise aapki repository mein changes karne ka access diya gaya hota hai.
